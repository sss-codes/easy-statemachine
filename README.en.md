# easy-statemachine

#### Description
Easy Statemachine是一个轻量级的、基于事件驱动的状态机框架，可以便捷的嵌入到Spring MVC / Spring Boot项目中，统一控制状态流转，避免随处编写状态更改语句造成后续难以修改扩展。
